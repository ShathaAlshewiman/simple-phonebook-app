# 📱 Phone Book Application

A simple Python-based phone book application that allows users to search, add, and manage contacts.

## 🚀 Features
- Search for a name using a phone number
- Search for a phone number using a name
- Add new contacts
- Input validation for phone numbers

## 🛠️ Technologies Used
- Python 3

## Notes
Phone numbers must be exactly 10 digits
Data is stored temporarily (not saved after closing the program)

## ▶️ How to Run
1. Make sure Python is installed
2. Run the program:

```bash
python PhoneBookProject.py
